## 專案簡介

本專案為 **C# Windows Forms 視窗程式設計** 的 BMI 計算機作業，主要功能是讓使用者輸入身高與體重後，快速計算 BMI（Body Mass Index，身體質量指數），並依照數值自動判定體位狀態。

除了基本計算功能外，本系統也額外加入了健康建議、BMI 說明、資料清除、快捷鍵操作、結果顏色變化、跑馬燈健康小語以及關於系統等功能，讓整體使用體驗更完整，也讓介面更具互動性與完成度。

本系統的特色如下：

- 可輸入身高（公分）與體重（公斤）進行 BMI 計算
- 具備輸入驗證，避免錯誤資料造成程式異常
- 依 BMI 結果顯示不同體位分類
- 以不同背景顏色呈現結果，讓判讀更直觀
- 保留跳出視窗提示，讓使用者明確看到結果與提醒
- 提供各體位對應的健康建議
- 提供 BMI 公式與分類標準說明
- 可一鍵清除資料，方便重新輸入
- 支援鍵盤快捷操作，減少滑鼠使用
- 加入健康小語跑馬燈，增加介面活潑度
- 提供「關於本系統」資訊視窗

## 功能說明

### 1. BMI 計算

使用者輸入：

- 身高（cm）
- 體重（kg）

系統會自動使用以下公式計算 BMI：

`BMI = 體重(kg) / 身高(m)^2`

並依照結果判定為：

- BMI < 18.5：體重過輕
- 18.5 ≤ BMI < 24：健康體位
- 24 ≤ BMI < 27：體位過重
- 27 ≤ BMI < 30：輕度肥胖
- 30 ≤ BMI < 35：中度肥胖
- BMI ≥ 35：重度肥胖

---

### 2. 輸入檢查

系統會先檢查使用者輸入的內容是否為合法數字，並確認身高與體重必須大於 0。

若輸入錯誤，會跳出提示視窗提醒使用者，例如：

- 請輸入正確的數字
- 身高與體重必須大於 0

這樣可以避免輸入異常值時造成錯誤計算。

---

### 3. 結果顯示

當 BMI 計算完成後，畫面會顯示：

- BMI 數值（小數點後兩位）
- 體位判定結果

同時系統會根據不同分類自動改變結果區塊的背景顏色，以及表單背景色彩，讓使用者更容易辨識目前的體位狀態。

此外，計算完成後也會保留跳出視窗提示，讓結果更明確。

---

### 4. 健康建議

系統會根據最近一次的 BMI 結果提供對應建議，內容包含：

- 飲食建議
- 運動建議
- 生活建議

例如：

- 體重過輕者建議增加均衡熱量攝取與肌力訓練
- 健康體位者建議維持規律運動與良好作息
- 體位過重或肥胖者建議控制熱量、增加有氧運動並注意健康管理

此功能讓本系統不只是單純計算工具，也能提供基本健康參考。

---

### 5. BMI 說明

系統提供 BMI 的基本介紹，內容包含：

- BMI 的用途
- 計算公式
- 體位分類標準
- BMI 的限制與注意事項

能幫助使用者理解 BMI 的意義，而不是只看到數值結果。

---

### 6. 清除資料

按下清除功能後，系統會：

- 清空身高欄位
- 清空體重欄位
- 清空結果顯示
- 重設結果顏色
- 將游標移回輸入欄位

方便使用者重新輸入資料。

---

### 7. 關於本系統

系統內建簡單的版本資訊與功能介紹，可讓使用者知道：

- 系統名稱
- 版本
- 開發工具
- 快捷鍵功能
- 作者資訊

---

### 8. 鍵盤快捷操作

本系統支援多種鍵盤操作，提升輸入效率：

- 在身高欄位按 `Enter`：自動跳到體重欄位
- 在體重欄位按 `Enter`：直接計算 BMI
- 按 `Esc`：清除資料
- 按 `F1`：顯示 BMI 說明
- 按 `F2`：顯示健康建議
- 按 `F3`：顯示關於本系統
- 按 `Alt + C`：執行 BMI 計算
- 按 `Alt + L`：清除資料
- 按 `Alt + X`：離開程式

這些設計可減少滑鼠操作，讓使用過程更流暢。

---

### 9. 健康小語跑馬燈

本系統在畫面下方加入健康小語跑馬燈，透過 `Timer` 控制文字持續移動，讓畫面不會過於靜態，也增加了視覺上的活潑感。

此功能屬於課堂基本內容之外的額外創意擴充，可展現程式設計上的完整度與巧思。
